# Noveler 敏感词库

## 📊 词库统计

- **总词数**：900 个
- **高危级别**：250 词（政治、严重暴力、严重违法）
- **中危级别**：350 词（色情、一般违法、宗教敏感）
- **低危级别**：300 词（广告、争议词汇、不文明用语）

> 📌 **版本**：v1.0.0 | **更新时间**：2025-12-03

## 🎯 词库分级说明

### 🔴 高危级别（High） - 强烈建议启用

**包含内容**：
- 政治敏感词汇（政党、政治事件、政治运动等）
- 颠覆性言论（推翻、暴动、政变等）
- 严重违法活动（暴力犯罪、严重社会危害等）

**风险说明**：这些词汇涉及政治和法律红线，极易导致作品被下架或账号被封，**强烈建议始终启用**。

### 🟠 中危级别（Medium） - 建议启用

**包含内容**：
- 色情内容（直接性描写、色情交易等）
- 毒品赌博（毒品种类、赌博场所等）
- 违法犯罪（诈骗、传销、走私等）
- 宗教敏感（宗教极端、特定宗教问题等）

**风险说明**：这些词汇容易触发平台审核机制，可能导致章节被锁或需要修改，**建议启用**。

### 🟡 低危级别（Low） - 可选启用

**包含内容**：
- 不文明用语（脏话、网络骂街等）
- 广告营销（推广词汇、诱导关注等）
- 争议词汇（地域/性别/种族歧视等）

**风险说明**：这些词汇主要影响内容质量和读者体验，不一定违规但建议避免，**可根据需要启用**。

## 🔧 自定义词库

### 创建自定义黑名单

如果需要添加额外的敏感词：

1. 在项目根目录创建 `.noveler/sensitive-words/` 文件夹
2. 复制 `templates/sensitive-words/blacklist-template.json` 到该文件夹
3. 重命名为 `blacklist.json`
4. 编辑 `words` 数组，添加您的敏感词

**示例**:
```json
{
  "description": "我的自定义黑名单",
  "words": [
    "我想屏蔽的词1",
    "我想屏蔽的词2",
    "平台特定敏感词"
  ]
}
```

### 创建自定义白名单

如果检测到误报（如人物名、地名与敏感词重复）：

1. 复制 `templates/sensitive-words/whitelist-template.json` 到 `.noveler/sensitive-words/`
2. 重命名为 `whitelist.json`
3. 添加误报词汇

**示例**:
```json
{
  "description": "我的自定义白名单",
  "words": [
    "主角名字李独",
    "地名天元城",
    "功法名无极心法"
  ]
}
```

## ⚙️ 配置方式

在项目的 `novel.json` 中配置：

```jsonc
{
  "noveler": {
    "sensitiveWords": {
      // 总开关
      "enabled": true,

      // 检测级别（可独立开关）
      "levels": {
        "high": true,       // ✅ 高危：强烈建议启用
        "medium": true,     // ✅ 中危：建议启用
        "low": false        // ❌ 低危：可选启用（默认关闭）
      },

      // 检测时机
      "checkOnType": true,        // 输入时检测（防抖 500ms）
      "checkOnSave": true,        // 保存时检测

      // 自定义词库
      "customWords": {
        "enabled": true,
        "blacklistPath": ".noveler/sensitive-words/blacklist.json",
        "whitelistPath": ".noveler/sensitive-words/whitelist.json"
      },

      // 显示设置
      "display": {
        "severity": "Warning",              // "Error" | "Warning" | "Information"
        "showInProblems": true,             // 在问题面板显示
        "showWordCount": true               // 状态栏显示敏感词数量
      }
    }
  }
}
```

## 💡 使用建议

### 按题材选择级别

| 题材类型 | 推荐配置 | 说明 |
|---------|---------|------|
| 🌐 现实题材 | `high + medium + low` | 涉及现实社会，需要全面检查 |
| ⚔️ 玄幻仙侠 | `high + medium` | 架空世界，中低危词汇可适当放松 |
| 🏙️ 都市言情 | `high + medium + low` | 贴近生活，建议全面检查 |
| 🚀 科幻未来 | `high + medium` | 架空设定，根据剧情调整 |
| 🎭 历史穿越 | `high + medium` | 涉及历史，需注意政治敏感 |

### 使用流程建议

1. **创作初期**：可以只启用 `high` 级别，避免影响创作思路
2. **初稿完成**：启用 `high + medium`，修改明显问题
3. **发布前**：启用所有级别，全面检查
4. **上架后**：根据平台反馈调整白名单

### Quick Fix 使用技巧

当检测到敏感词时，可以通过 Quick Fix（灯泡💡）：
- **添加到白名单**：确认是人名/地名等合理使用
- **删除词汇**：直接删除敏感词
- **替换为星号**：用 *** 代替（适用于引用、对话等场景）
- **忽略此处**：暂时忽略，稍后处理

## ⚠️ 重要声明

1. **词库仅供参考**：不同平台审核标准不同，本词库无法保证 100% 覆盖
2. **上下文很重要**：某些词汇在特定上下文中可能不算敏感（如"革命"在历史题材中）
3. **最终审核以平台为准**：词库只是辅助工具，最终是否通过审核由各平台决定
4. **建议人工复核**：自动检测可能有误报，发布前建议人工再次确认
